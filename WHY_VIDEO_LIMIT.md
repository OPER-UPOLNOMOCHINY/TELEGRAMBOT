# Почему бот не может обработать большое видео?

## Проблема:

Telegram Web App API имеет **жесткое ограничение** на размер данных, которые можно отправить через метод `sendData()`:

- **Максимальный размер: ~64KB** (65,536 байт)
- Это ограничение самого Telegram, его нельзя обойти

## Почему это ограничение:

1. **Безопасность**: Предотвращает перегрузку серверов Telegram
2. **Производительность**: Быстрая передача данных через Web App
3. **Архитектура**: `sendData()` предназначен для небольших данных (текст, JSON)

## Решения:

### ✅ Решение 1: Уменьшить размер видео (текущее)
- Длительность: 1-1.5 секунды
- Разрешение: 240x180
- Битрейт: 200 kbps
- FPS: 10-15

**Результат**: Видео ~20-40 KB - помещается в лимит

### ✅ Решение 2: Отправить видео вручную
Если видео слишком большое:
1. Сохраните видео на устройство
2. Отправьте его вручную в чат с ботом
3. Бот обработает его как обычное видео

### ✅ Решение 3: Использовать альтернативный способ (сложнее)
- Отправка через Bot API напрямую (требует сервер)
- Использование облачного хранилища
- Разбиение на части

## Текущая реализация:

Бот автоматически:
- Пытается отправить маленькое видео через `sendData()`
- Если слишком большое - просит отправить вручную
- Обрабатывает видео, отправленное вручную в чат

## Рекомендация:

**Оставьте текущие настройки** (1-1.5 сек, низкое качество) - это оптимальный баланс между:
- Размером файла (< 64KB)
- Качеством для детекции фейков
- Скоростью обработки
